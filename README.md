# Agent Skills

General-purpose skills for AI coding agents, following the [Agent Skills](https://agentskills.io) open standard.

## Skills

| Skill | Description |
| --- | --- |
| **fix-tests** | Diagnose and fix CI test failures from CircleCI. Finds the most recent pipeline on the current (or specified) branch and methodically fixes failing tests. |
| **make-skill** | Create a new agent skill through an interactive interview process. |

## Setup

Run `setup.sh` to symlink skills into `~/.claude/skills/`. See the script for details on multi-repo support.
